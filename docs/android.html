Android SDK

GrowingIO Autotracker Android SDK 可以通过 如何集成教程 来集成SDK，SDK同时具备埋点和无埋点功能：

  • 埋点功能：只自动采集用户访问事件，需要开发同学调用相应埋点 API 采集埋点事件;
  • 无埋点功能：具备自动采集基本用户行为事件，如页面访问，点击事件等。
