  • 版本记录

4.3.2 (2024-07-26)

Bug Fixes​

  • 移除编译产物中的 NSLog 符号 (074e9cd)
  • 在 Flutter 应用中，当 Session 状态未处于 active 时，不发送 PAGE 事件 (de4b5bc)

4.3.1 (2024-05-31)

Bug Fixes​

  • visionOS 平台使用 Cocoapods 集成支持，以及稳定性优化 (#317) (9ab87ec)

4.3.0 (2024-05-07)

Bug Fixes​

  • 优化事件发送逻辑 (#304) (d54bef0)
  • 添加 privacy manifest (watchOS/tvOS/visionOS 平台) (#303) (3ce88fc)
  • autotrack: 当配置 autotrackAllPages 为 true 时，忽略 GioKit 页面 (#302) (59e5cdd)
  • perf: 优化函数 growingHelper_beautifulJsonString 造成的性能损失 (#312) (f8150cb)
  • 修复 UICollectionView/UITableView 点击监听 bug (#316) (0ec1ab0)
  • v2Adapter: 兼容更多老 SaaS 的属性值的类型 (#307) (f2f032b)

Features​

  • 新增动态通用属性接口 (#308) (bec7290)
  • 新增配置项 dataValidityPeriod (#310) (a52fefa)
  • 新增 watchOS/tvOS/visionOS/SwiftUI 支持 (#300) (0ef1b26)

4.2.0 (2024-03-14)

Features​

  • autotrack: 新增配置项 autotrackAllPages (#301) (5cb409b)

4.1.0 (2024-01-25)

Features​

  • 新增 App Extension 支持 (#296) (4a9d038)
  • 更新 Flutter 模块逻辑 (#295) (a52bc94)

4.0.0 (2023-12-25)

Features​

  • 新增 AB 实验模块 (#281) (2d3950f)
  • 新增通用属性相关接口 (#286) (e60f1fa)
  • 无埋点重构 (#268)

更多发布细节请参考 SDK Releases in Github​
