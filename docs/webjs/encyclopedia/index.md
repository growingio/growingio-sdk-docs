文本主要内容更多的是介绍 WebJS SDK 的实现细节和扩展，帮助您了解一些实现层面逻辑。通过本文内容，希望你能了解如何能更好的利用 WebJS SDK 优雅并精准地采集数据和建立企业的内部数据指标体系。

主要内容为：

* **第一章、介绍**

从宏观角度和一定的技术角度说明我们的SDK如何采集数据。

* **第二章、元数据**

解释我们采集的数据都是从哪些维度采集，我们又可以利用这些维度做怎样的分析。

* **第三章、指标规则**

详细介绍和解释无埋点的采集指标规则。

* **第四章、高阶功能**

在基础的采集能力上，如何更高级地控制采集和利用采集能力。
