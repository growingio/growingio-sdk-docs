SDK调试

在Web中调试SDK，若你使用未压缩的包可以直接通过debugger的形式来调试，若只是想看数据可以直接在初始化时将debug配置项设置为true即可，如下：

gdp('init', 'your projectId', 'your dataSourceId', {  host: 'your apiServerHost',   version: '1.0.0',  debug: true  // 打开debug调试能力});
