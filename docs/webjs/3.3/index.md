title: Web JS SDK（3.3）
---


Web JS SDK3.0 将H5内嵌页，Hybrid SDK，Web JS SDK集成与一体，具备以下特性：

* 埋点能力，开发同学调用API主动采集埋点事件
* 无埋点能力，自动采集用户行为事件，可通过开关控制
* 小程序打通，方便于小程序打通访问用户
* 原生打通，方便于原生App打通访问用户
* Web可视化圈选支持
* Hybrid可视化圈选支持

:::caution
**注：WebJS SDK 3.3版本已不再维护，请尽快升级至3.8或4.0版本，具体升级步骤请咨询我们的技术支持。**
:::
