## [1.1.0](https://github.com/growingio/growingio-sdk-harmonyos/tree/1.1.0) (2024-06-21)

### Features

* 适配 HarmonyOS NEXT
* 支持从 OpenHarmony API 10 到 API 12
* 支持 New SaaS/SaaS/CDP 平台集成
* 支持多实例采集
* 支持通用属性
* 支持 hybrid 打通
* 支持事件数据加密压缩上报
* 新增初始化配置项，如：网络请求超时时长、数据有效时长、加密、压缩等等

### Bug Fixes

* 优化 SDK 初始化方式，见 README
* 其他稳定性优化
